Firebase Kurulumu Gerekiyor

Google giris icin Firebase projesi olusturmaniz gerekiyor. Asagidaki adimlari takip edin:

  1. console.firebase.google.com adresine gidin
  2. "Add project" ile yeni proje olusturun
  3. Sol menu: Authentication → Get started → Google provider'i aktif edin
  4. Sol menu: Firestore Database → Create database → Test mode secin
  5. Sol menu: Project Settings → "Add app" (web </> ikonu) → Config kodunu kopyalayin
  6. Authentication → Settings → Authorized domains → ikoncu.github.io ekleyin
  7. Bu HTML dosyasinin basindaki FIREBASE_CONFIG objesini doldurun
